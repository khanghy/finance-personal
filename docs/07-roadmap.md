# Roadmap

## MVP checklist

- Scaffold Next.js, Tailwind, shadcn/ui-style primitives.
- Supabase Auth magic link.
- Initial schema và RLS.
- Dashboard tổng quan.
- Giao dịch CRUD và CSV import/export.
- Nợ CRUD, thanh toán nợ, avalanche/snowball.
- In-app notification.
- AI advisor adapter và lưu lịch sử phiên.
- Unit tests cho domain logic.
- Vercel env checklist.

## Post-MVP

- Email notification qua Resend.
- Push notification.
- Recurring transactions nâng cao.
- Budget theo category.
- Multi-currency.
- Family/shared workspace.
- Bank integration nếu có provider phù hợp.
- PWA offline-lite.

## Backlog ưu tiên

1. Hoàn thiện auth và RLS bằng Supabase thật.
2. CRUD giao dịch/nợ production-ready.
3. Import CSV có mapping cột linh hoạt.
4. AI advisor dùng dữ liệu thật.
5. Reminder scheduler qua Vercel Cron hoặc Supabase Edge Function.
